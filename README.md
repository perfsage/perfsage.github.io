## About

PerfSage is a micro-business and knowledge hub focused on **performance engineering productivity using AI**. We build open-source JMeter plugins, tools, and content to help performance engineers work smarter.

## What We Do

- **Open Source JMeter Plugins** - Free tools to enhance your performance testing workflow
- **AI-Powered Insights** - Automatic test result analysis and anomaly detection
- **Performance Engineering Blog** - Guides, best practices, and tips for modern performance engineers
- **Community** - Sharing knowledge with performance engineering professionals worldwide

## Coming Soon
